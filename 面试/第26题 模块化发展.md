## CommonJs: 模块规范

**实现: node(服务端)、webpack(浏览器)、browserfy(浏览器)**

基本语法: 

1. 引入: 
   * 第三方: var fs = require('fs');
   * 自定义模块 require('./xxx.jpg');
   * 读入并执行一个 JavaScript 文件, 然后返回该模块的 exports 对象, 如果没有发现指定模块, 会报错
2. 暴露: 
   * module.export = value;
   * export.xxx = value;
   * module 变量代表当前模块. 这个变量是一个对象, 它的 exports 属性(即 module.exports ) 是对外的接口
