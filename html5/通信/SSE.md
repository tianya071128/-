# SSE(Server-sent events)

**SSE(Server-Sent Events, 服务器发送事件): 是围绕只读Coumet 交互推出的API 或者 模式**

**用于创建到服务器的单向连接, 服务器通过这个连接可以发送任意数量的数据**

**在 Web 应用程序中使用服务器发送事件很简单, 在服务器端, 只需要按照一定的格式返回事件流, 在客户端中, 只需要为一些时间类型绑定监听函数**
