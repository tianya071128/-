## 1. 自适应的椭圆

```css
	/* 椭圆 */
    .ellipse {
      border-radius: 50%;
    }

    /* 半椭圆 */
    .half-ellipse {
      border-radius: 50% / 0 0 100% 100%;
    }

    /* 四分之一椭圆 */
    .quarter-ellipse {
      border-radius: 100% 0 0 0;
    }

    /* 八分之一椭圆、三分之一椭圆 是无法生成的 */
```
