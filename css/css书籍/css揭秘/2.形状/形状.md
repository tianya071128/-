```css
	/* 椭圆 */
    .ellipse {
      border-radius: 50%;
    }

    /* 半椭圆 */
    .half-ellipse {
      border-radius: 50% / 0 0 100% 100%;
    }

    /* 四分之一椭圆 */
    .quarter-ellipse {
      border-radius: 100% 0 0 0;
    }

    /* 八分之一椭圆、三分之一椭圆 是无法生成的 */
```

## 2. 平行四边形

```css
/* 嵌套元素方案: 对内容再应用一次反向的skew() 变形，从而抵消容器的变形效果 */
.parallelogram {
  transform: skewX(-45deg);
}

.parallelogram span {
   display: inline-block;
   transform: skewX(45deg);
}

 /* 伪元素方案 */
.pseudo-parallelogram {
   background-color: transparent;
   position: relative;
}

.pseudo-parallelogram::before {
   content: '';
   position: absolute;
   top: 0;
   bottom: 0;
   left: 0;
   right: 0;
   background: #6999c1;
   transform: skewX(-45deg)
   /* 使伪元素堆叠层次被推到宿主元素之下 */
   z-index: -1;
}
```

## 3. 菱形图片

```css
	/* 基于变形的方案, 缺点在于: 只能处理正方形 */
    .diamond {
      transform: rotate(45deg);
      overflow: hidden;
      border: 1px solid #ccc;
    }

    .diamond-son {
      width: 100%;
      height: 100%;
      display: inline-block;
      background-color: #6999c1;
      /* rotate(-45deg) 用来抵消父元素的旋转, 但是此时会形成一个 八边形 */
      transform: rotate(-45deg);
    }

    .scale {
      /* scale 缩放形成菱形 */
      transform: rotate(-45deg) scale(1.42);
      line-height: 200px;
    }

    /* 裁切路径方案: clip-path(浏览器支持度不高) */
    .cutting {
      background-color: #6999c1;
      clip-path: polygon(50% 0, 100% 50%,
          50% 100%, 0 50%);
      transition: 1s clip-path;
    }

    /* 还可以用来参与动画 */
    .cutting:hover {
      clip-path: polygon(0 0, 100% 0,
          100% 100%, 0 100%);
    }
```

## 4. 切角效果

```css
	.cutting {
      /* 四个切角, 四种渐变色 */
      background:
        linear-gradient(135deg, transparent 15px, #58a 0) top left,
        linear-gradient(-135deg, transparent 15px, #58a 0) top right,
        linear-gradient(-45deg, transparent 15px, #58a 0) bottom right,
        linear-gradient(45deg, transparent 15px, #58a 0) bottom left;
      /* 将渐变色形成的图片尺寸缩小 */
      background-size: 50% 50%;
      /* 不平铺渐变图片背景 */
      background-repeat: no-repeat;
    }

    /* 弧形切角, 利用径向渐变 */
    .arc {
      background: #58a;
      background:
        radial-gradient(circle at top left,
          transparent 15px, #58a 0) top left,
        radial-gradient(circle at top right,
          transparent 15px, #58a 0) top right,
        radial-gradient(circle at bottom right,
          transparent 15px, #58a 0) bottom right,
        radial-gradient(circle at bottom left,
          transparent 15px, #58a 0) bottom left;
      background-size: 50% 50%;
      background-repeat: no-repeat;
    }

	/* 内联 SVG 与 border-image 方案 */
    .svg {
      border: 21px solid #58a;
      border-image: 1 url('data:image/svg+xml,\
	                      <svg xmlns="http://www.w3.org/2000/svg" width="3" height="3" fill="%2358a">\
	                      <polygon points="0,1 1,0 2,0 3,1 3,2 2,3 1,3 0,2" />\
	                      </svg>');
      background: #58a;
      background-clip: padding-box;
      transition: border-width 1s;
    }

    /* 实现动画, 但是有 bug */
    .svg:hover {
      border-width: 30px;
    }

    /* 裁切路径方案 */
    .caiqie {
      clip-path: polygon(20px 0, calc(100% - 20px) 0, 100% 20px,
          100% calc(100% - 20px), calc(100% - 20px) 100%,
          20px 100%, 0 calc(100% - 20px), 0 20px);
    }
```
